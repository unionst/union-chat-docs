# UnionChat

A complete, stubbed implementation of a modern chat SDK API for documentation and reference purposes.

## Features

## Structure

- **Sources/UnionChat/** - Complete API organized into categories
  - All symbols properly stubbed for compilation
  - Each file contains main type declaration + all extensions merged

- **Documentation.docc/** - Documentation resources (images, guides)

## Documentation

All symbols in UnionChat should have comprehensive documentation comments following the guidelines in [DOCUMENTATION_STYLE_GUIDE.md](DOCUMENTATION_STYLE_GUIDE.md).

Key principles:
- Comprehensive coverage (what/where/when/why/how)
- Real-world context and decision-making guidance
- Focused examples with modern Swift syntax (iOS 17+)
- Complete parameter documentation
- At least one clear example per symbol
